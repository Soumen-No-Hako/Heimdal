# Heimdal 

The idea of Heimdal is - Me as a user will be able to easily find words withing my file system.

## Reason

We all know windows may become a headache in terms of searching files. And the UI only searches by name
So I decided to make my own solution

## Challenge/ Goals

* Configure file locations
* Loading large files without memory leaks
* Use **Inverted Index** technique to fast look up word locations
* Minimise search miss
